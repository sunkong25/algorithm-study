# Algorithm Study

코딩 테스트 대비를 위한 알고리즘 스터디 레포지토리입니다.
문제 풀이와 코드 리뷰를 통해 다양한 알고리즘 패턴을 학습하는 것을 목표로 합니다.

---

# 🎯 Study Goal

* 알고리즘 문제 해결 능력 향상
* 다양한 알고리즘 패턴 학습
* 코드 리뷰를 통한 사고 과정 공유
* 지속적인 문제 풀이 습관 형성

---

# 📅 Study Rule

* **주 2회 진행**
* **회차당 3문제 풀이**
* 사전에 공지된 문제 풀이 후 **Pull Request 제출**
* 스터디 시간에 풀이 방식 및 알고리즘 토론 진행

---

# 📁 Repository Structure

```text
algorithm-study

problems
   week01-session01
      01_problem.md
      02_problem.md
      03_problem.md

solutions
   week01-session01
      A
      B
      C
      D
```

### problems

문제 링크 및 문제 정보 정리

### solutions

각 스터디원의 문제 풀이 코드

---

# 🧑‍💻 How to Solve

각 문제 풀이 시 다음 내용을 정리합니다.

1. 문제 링크
2. 알고리즘 유형
3. 접근 방법
4. 시간 복잡도 또는 풀이 시간
5. 기억하고 싶은 포인트

---

# 🔀 PR Rule

### PR Title

```text
[week01-session01] 이름
```

예시

```text
[week01-session01] eddy
```

---

### PR Description

PR에는 아래 내용을 포함합니다.

* 체크리스트 형태의 solved problems
* 문제별 접근법
* 문제별 기억하고 싶은 점

시간 복잡도 계산이 어려운 경우에는 풀이 시간, 실행 시간, 또는 간단한 사유로 대체할 수 있습니다.

---

# 💬 Code Review

코드 리뷰 시 다음 내용을 중심으로 확인합니다.

* 더 효율적인 알고리즘 존재 여부
* 접근 방식의 타당성
* 코드 가독성
* 다른 풀이 아이디어

---

# 📌 Commit Convention

```text
week01-session01: 문제 풀이 제출
```

예시

